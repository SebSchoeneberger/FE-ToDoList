# üìù To-Do App (Frontend)
Frontend for a simple To-Do application built with **React + TypeScript**, powered by a **Payload CMS REST API** backend.

This app allows authenticated users to:
- Create To-Dos  
- View To-Dos  
- Edit To-Dos  
- Delete To-Dos  
- Mark as completed  
- Filter by **Category** and **Status**  
- Manage Categories
